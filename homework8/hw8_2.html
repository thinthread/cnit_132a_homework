Brief overview of GA4 Event Report for this site

(Event Report focuses user(s) site visits)


At the beginning of the CNIT 132A Advanced HTML and CSS course, students were introduced to and explored Google Analytics (GA4) in order to complete Homework Assignment 2.


Homework Assignment 2/GA4 provided the user with a range of reports and tools that had been used to collect meaningful data, which can then be utilized in the creation and maintenance of the most effective website.



Here are the results of the data collected over 28 days for this website via the GA4 Events Report.





The graph in this report displays the following data event categories: Page_View, Scroll, User_Engagement, Session_Start, and First Data Collected.






In this section of the report, we see data that recorded the following activities: Page_View = 82, Scroll = 82, User_Engagment = 61, Session Start = 23, and First_Visit = 7.


Whilst this project may be a relatively small endeavor, I found the above mentioned data collection points very interesting, particularly that of the Total Users = 11. I also found it of interest that I had given out my website to 5 people in total, including the professor.
